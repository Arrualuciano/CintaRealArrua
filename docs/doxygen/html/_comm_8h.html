Descripción detallada

Modulo de comunicacion serie con protocolo UNER.

Implementa el protocolo de trama UNER sobre UART0 del ATmega. Formato de trama:

[ 'U' | 'N' | 'E' | 'R' | LEN | ':' | CMDID | DATA... | CKS ]

donde LEN = 1 (CMDID) + N (bytes de datos) + 1 (CKS), y CKS = XOR de todos los bytes desde 'U' hasta el ultimo dato.

El modulo utiliza dos buffers circulares (RingBuf) de 128 bytes para desacoplar la ISR de recepcion del procesamiento en el main loop.

◆ Comm_Process()

void Comm_Process ( Comm_Protocol * comm)

Procesa el modulo de comunicacion: envio y recepcion.

Debe llamarse periodicamente desde el main loop (no bloqueante).

  • TX: transmite un byte pendiente si el registro UDR0 esta libre.
  • RX: corre la FSM UNER sobre todos los bytes disponibles en el buffer de recepcion e invoca el callback onCmd al completar una trama valida con checksum correcto.
Parámetros
commPuntero a la instancia Comm_Protocol.
