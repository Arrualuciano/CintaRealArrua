Descripción detallada

Clasificador de cajas para cinta transportadora (modo open-loop).

Mantiene una lista de cajas en transito. Cada caja tiene asignado un servo destino y un timer de arribo (timePushBox). El timer se decrementa cada 2 ms en Classifier_Process(); al llegar a cero se dispara el servo correspondiente.

Flujo de uso:

Classifier_Init(&clas, boxTypes, onTrigger);
Classifier_NewBox(&clas, tipo); // al medir una caja
Classifier_Process(&clas); // cada 2 ms

◆ Classifier_Process()

void Classifier_Process ( classifier_t * clas)

Decrementa los timers de las cajas en transito y dispara los servos.

Debe llamarse cada 2 ms desde el tick del Timer0. Recorre la lista de cajas, decrementa timePushBox y al llegar a 0 invoca onTrigger() con el servo correspondiente.

Parámetros
clasPuntero al clasificador.

Debe llamarse cada 2 ms desde el tick del Timer0. Recorre la lista de cajas, decrementa timePushBox y al llegar a 0 invoca onTrigger() con el servo correspondiente. La caja disparada se elimina de la lista.

Parámetros
clasPuntero al clasificador.
