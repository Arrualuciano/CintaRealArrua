Descripción detallada

Implementacion del driver PWM por software para servomotores SG90.

Usa Timer2 en modo CTC (prescaler 64, OCR2A = 124) para generar una interrupcion cada 0.5 ms. Un contador ciclico de 40 ticks produce el periodo de 20 ms (50 Hz) requerido por los servos SG90.

En cada ciclo:

  • tick 0: se activan las salidas de los servos habilitados.
  • tick N (= ancho de pulso): se desactiva la salida correspondiente.

Mapeo de pines (segun consigna):

  • Servo 0 (SERVO 1): PD7
  • Servo 1 (SERVO 2): PB3
  • Servo 2 (SERVO 3): PB4

ISR ( TIMER2_COMPA_vect )

ISR de Timer2 Compare Match A: genera la senal PWM de los servos.

Se ejecuta cada 0.5 ms. Incrementa el contador ciclico y controla los pines de salida segun el ancho de pulso configurado.
